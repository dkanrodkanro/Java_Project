# Java 기초 학습 기록

이 저장소는 Java 기초 문법을 다시 학습하며  
코드와 커밋으로 학습 과정을 기록하기 위한 저장소입니다.

## 학습 목적
- Java 기본 문법 복습
- Git / GitHub 사용에 익숙해지기
- 추후 백엔드(Spring) 학습을 위한 기초 다지기

## 학습 방식
- 온라인 강의(생활코딩 Java 입문)를 참고
- 각 주제별로 직접 예제 코드를 작성
- 주제 단위로 커밋을 나누어 기록

## 현재 학습 내용
- 숫자와 문자열
- 조건문 (if / else, switch)
